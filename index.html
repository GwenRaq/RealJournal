UX stands for User Experience. It enhances the inability between the user and a product. It makes the product more fun to use and interactable. It is what the user will interact with. It also requires organization.

UI stands for User Interface. It is the graphical design of the site, the visual element, and what you will see in the site.

Responsive and Mobile Design

October 7, 2016

Responsive Design

A responsive design is building a site to respond to multiple sizes also known as fluid design. So when you make the window smaller, the website will also become smaller to fit the window. It won't cut of any of its properties. An example of it would be Facebook, Google, and beta.rallyinteractive.com.

Mobile Design

A mobile design is a web app design for mobile devices. Media queries makes the design effective for a mobile. An example is Instagram. A Media Query is break points in css that tells your app to change according to screensize.
